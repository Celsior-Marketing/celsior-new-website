# Content Control Pilot

This folder contains temporary content-control files for the Celsior static website.

Purpose:
- Support approved Google Sheet exports.
- Validate content changes locally before applying them.
- Keep Adithya as final gatekeeper before commit/deploy.
- Avoid direct publishing from Google Sheets.

Current pilot area:
- Homepage rolling news cards.

Important:
- This is not a CMS.
- This should not replace the future WordPress build.
- Scripts should update only clearly defined pilot areas.
