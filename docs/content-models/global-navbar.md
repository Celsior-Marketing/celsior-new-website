# Content Model: Global Navbar

## Purpose

Define the future CMS-editable structure for the site navbar.

## Top-Level Items

Each top-level item should support:

- Label
- Dropdown enabled
- Dropdown groups
- Child links
- Display order

Top-level navbar labels should act as dropdown triggers, not direct links.

## Child Link Fields

Each dropdown link should support:

- Label
- URL
- Group/category
- Optional badge
- Open in new tab: yes/no
- Status: active / pending / hidden

## Notes

Pending pages should not be removed until confirmed.
