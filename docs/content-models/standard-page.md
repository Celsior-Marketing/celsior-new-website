# Content Model: Standard Page

## Recommended Fields

### Page Metadata

- Page title
- SEO title
- SEO description
- Open Graph image
- Canonical URL

### Hero

- Eyebrow
- Headline
- Subheadline/body copy
- Primary CTA label
- Primary CTA URL
- Secondary CTA label
- Secondary CTA URL
- Background image or visual

### Content Sections

Each section should support:

- Section type
- Eyebrow
- Heading
- Body copy
- Cards/items
- CTA label
- CTA URL
- Image/background URL
- Notes

### CTA Block

- Heading
- Body copy
- Button label
- Button URL
- HubSpot form/modal type if applicable
