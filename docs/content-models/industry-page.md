# Content Model: Industry Page

## Recommended Fields

### Page Metadata

- Industry name
- SEO title
- SEO description
- Canonical URL

### Hero

- Eyebrow
- Headline
- Subheadline
- Primary CTA label
- Primary CTA URL
- Background image

### Pain Points / Challenges

Each item:

- Title
- Description
- Icon/image

### Capabilities / Solutions

Each item:

- Title
- Description
- URL
- Optional CTA

### Proof / Case Studies

Each item:

- Title
- Client/industry label
- Summary
- URL
- Image

### Final CTA

- Heading
- Body copy
- Button label
- Button URL
